## Projects

### KubeLens AI

Kubernetes operations and security platform.

Designed for cluster visibility, deterministic diagnostics, security posture checks, and AI-assisted root-cause explanation.

```text
Kubernetes · Go · React · FastAPI · Helm · Kustomize · OIDC · OpenTelemetry · Prometheus · Grafana
```

### SwarmLens

Operations console for Docker Swarm.

Built around live inventory, service diagnostics, secure operational actions, approvals, audit logs, and production visibility.

```text
Docker Swarm · Go · React · TypeScript · FastAPI · PostgreSQL · Redis · OIDC · Grafana
```

### StateSight

GitOps forensic platform.

Compares desired Git state with live Kubernetes state, groups drift into incidents, scores severity, and suggests actions.

```text
Kubernetes · GitOps · Go · React · PostgreSQL · Redis · REST APIs · Docker Compose
```
